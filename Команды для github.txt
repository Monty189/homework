Как обновлять данные в локальном репозитории:

git fetch обновляет на машине уже существующие файлы(не вызывает конфликтов)
git pull подтягиевает созданные файлы на github(и сливает ветки воедино ип часто вызывает конфликты)


что бы клонировать реп. на машину - с github нужно ввести команду git clone

что бы сохранить изменение git push(обновляет удаленный репозиторий)


чтобы клонировать репозитогрий через ссылку:
терминал вывести на рабочий стол и git clone и ссылку на него из гитхаба(берется в профиле, не в поисковой строке)
